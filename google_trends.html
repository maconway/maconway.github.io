Google Trends SU & SUD Stigma Code

This page provides supporting code for the paper Conway, Citrenbaum, Chen.  Substance use-related stigma:  an exploratory study of search behavior using Google Trends (2004-2021).  American Journal of Drug & Alcohol Abuse (2022).

An extended preprint version of the publication is available here.

The relevant code is organized into Jupyter Notebooks:

  • addict notebook
  • alcoholic notebook
  • junkie notebook
  • stoner notebook
  • alcohol use & alcohol abuse notebook
  • opioid use & opioid abuse notebook
  • substance use & substance abuse notebook
